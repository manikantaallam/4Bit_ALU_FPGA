# 4Bit_ALU_FPGA

This is 4bit ALU 

This is 4bit ALU 

Can do the following things 
OPC      Operations
---      -----------
1  ----> No Operation
2  ----> Comparision
3  ----> Or Logic
4  ----> Shift Left
5  ----> Rotate Right
6  ----> Addition
7  ----> Subtraction
8  ----> Multiplication

Synthesised and tested on: partan 3 series boards & Lattice ICE40-HX8K Breakout FPGA Boards
